Game Concept
Feline Frenzy is a game where you play as Chonkers the cat hero, navigating an abandoned laboratory, 
on a mission to rescue your lost friend while outsmarting robotic guards and uncovering hidden mysteries within the lab.

Input keys: 
In Main Menu scene: 
- Press "Play Game" button to go to the gameplay scene
- Press "How To Play" button to view tutorial
- Press "Quit Game" button to exit the game

Controls:
Player Movement - WASD
Shoot - Left Click
Change Element - E (When in range of elemental source)
Pause Game - Esc
