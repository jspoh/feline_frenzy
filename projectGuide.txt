Project Settings:
    Debug Mode for project:

    - Able to see systems' time consumption per game loop, in %. 

    Release Mode for project:

    - Able to see 2.5k object rendering with 60 FPS in only release mode.

To toggle between scenes: 
- Press 'Enter' key to go to the next scene

- Press 'C' key to crash the game and log the crash in src/Managers/crash_log.txt

Total of 3 scenes: 
- Animation scnene (press space button to show the animations)
- Collision scene (Left mouse click to clone entities)
- Performance scene (Left click to clone 500 entities at one click)
